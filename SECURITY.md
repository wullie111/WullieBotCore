# Security Policy

## Supported Versions
branches dev and main are supported 


| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability
create an issue describing the vulnerability

it will be fixed when the next update is released
